Table Row Disabler

Chrome Extension that allows user to lock input fields on specific table rows. The extension adds a lock button to each table row allowing the user to toggle the locked setting and remembers the state for return visits to the same website. The purpose is to help ensure you don't accidentally change table data that you are not supposed to change.

JS • HTML • CSS • Webpack • ESLint

Matching Game

A flip card matching game, more commonly known as 'Concentration'. The player must uncover all 8 pairs of cards consecutively one pair after the other. If the player makes a mistake or guesses wrong all cards flip back over requiring the player to start over.

Paintballnet Hotkeys

An advanced Bookmarklet that allows users to create hotkeys for the online game painballnet.net. Started as a proof of concept and transitioned into a full-fledged MVC / CRUD (create/read/update/delete) application written in vanilla JS. Uses a ES6 Proxy object to intercept the games WebSocket connection and injects a custom user interface that uses localstorage to persist settings.

JS • HTML • CSS • Webpack/Babel • WebSocket
